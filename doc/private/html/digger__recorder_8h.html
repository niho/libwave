drError drPauseRecording ( )

Pauses recording, i.e prevents the audio thread from passing audio data to the encoder until drResumeRecording() is called. If the system is not recording or if recording is already paused, this function does nothing.

drError drStartRecording ( const char *  audioFilePath)

Starts recording audio to a file at a given path. If audioFilePath already exists, it is assumed that some audio has already been recorded to that file and new audio will be appended to the end of the file.

Parameters
audioFilePathThe path of the file to record to.
